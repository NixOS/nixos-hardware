# Minisforum UM690

Mini PC with AMD Ryzen 9 6900HX (Rembrandt, Zen 3+) and Radeon 680M integrated graphics.

## Features

- AMD Ryzen 9 6900HX processor
- AMD Radeon 680M integrated GPU
- NVMe SSD support
- USB4/Thunderbolt support
- 2.5GbE Ethernet (Intel I225-V)
- Wi-Fi 6E (MediaTek MT7921K)
