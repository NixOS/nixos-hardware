# Minisforum UM790 Pro

Mini PC with AMD Ryzen 9 7940HS (Phoenix, Zen 4) and Radeon 780M integrated graphics.

## Features

- AMD Ryzen 9 7940HS processor
- AMD Radeon 780M integrated GPU (RDNA 3)
- NVMe SSD support
- USB4/Thunderbolt support
- 2.5GbE Ethernet (Realtek RTL8125)
- Wi-Fi 6E (Intel AX210)
